Décathlon - 110 mètres haies

Règles pour jouer au jeu le 110 mètres haies :

Vous commencez par lancer les 5 dés (en cliquant le bouton Allez! ) dont la somme des valeurs sera votre score. Si vous n'êtes pas satisfait du résultat, vous avez le droit de les relancez
Attention! Vous n'avez que 5 chances de relancez et votre nombre de lancer sera aussi important lors du classement finale.
